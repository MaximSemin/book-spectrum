# book-spectrum · спектральный анализ литературных произведений

Скрипт на Python строит **«спектр» книги**: загружает текст произведения по
ссылке, делит его на «страницы» (одна страница = заданное число слов), считает
на каждой странице упоминания цветов и рисует две диаграммы:

- **Спектр по ходу текста** — последовательность цветовых полос от начала к концу
  книги (каждая полоса = одна страница, её цвет = смесь упомянутых цветов,
  яркость = «цветность» страницы);
- **Сводное распределение** — доля каждого цвета во всём тексте.

Результат — самодостаточная HTML-страница (графика встроена как SVG),
публикуемая через **GitHub Pages**.

## 🔗 Результат

После включения Pages результат доступен по адресу:
**https://maximsemin.github.io/book-spectrum/**

Пример (значение по умолчанию): Ф. М. Достоевский, «Братья Карамазовы».

## Запуск локально

Зависимостей нет — нужен только Python 3.8+ (используется стандартная библиотека).

```bash
python spectrum.py "<ссылка на произведение>" \
    --words-per-page 300 \
    --output docs/index.html \
    --title "Автор. Название"
```

Пример:

```bash
python spectrum.py "http://az.lib.ru/d/dostoewskij_f_m/text_0100.shtml" \
    -w 300 -o docs/index.html -t "Ф. М. Достоевский. Братья Карамазовы"
```

### Параметры

| Параметр | По умолчанию | Назначение |
|---|---|---|
| `url` (позиционный) | — | ссылка на текст (например, страница az.lib.ru) |
| `--words-per-page`, `-w` | `300` | среднее число слов на одной странице |
| `--output`, `-o` | `docs/index.html` | путь к HTML-файлу результата |
| `--title`, `-t` | URL | заголовок страницы |

## Публикация через GitHub Pages

В репозитории есть workflow `.github/workflows/pages.yml`, который собирает и
публикует содержимое каталога `docs/`.

1. Откройте **Settings → Pages** и в разделе *Build and deployment* выберите
   **Source: GitHub Actions** (workflow пытается включить это автоматически при
   первом запуске).
2. Любой `push` в `main` пересобирает и публикует сайт.
3. Чтобы построить спектр **другого** произведения прямо в облаке, откройте
   вкладку **Actions → Build & Deploy spectrum to Pages → Run workflow** и
   укажите ссылку, число слов на страницу и заголовок.

## Как считаются цвета

- Текст приводится к нижнему регистру, «ё» → «е».
- Учитываются русские названия 12 цветов (красный, оранжевый, жёлтый, зелёный,
  голубой, синий, фиолетовый, розовый, коричневый, серый, чёрный, белый) во всех
  падежных и родовых формах прилагательных, а также характерные глаголы
  («покраснел», «побледнел», «пожелтел» …) и существительные цвета («румянец»,
  «синева», «седина» …).
- Совпадения по основе ограничены допустимыми окончаниями, поэтому слова вроде
  «прекрасный», «серьёзный», «серебро», «синтаксис», «голубчик» **не** считаются
  за цвет.
- Учитывается оттенок (hue); «тёмный» и «светлый» как обозначения яркости
  намеренно не включены.

Словарь цветов и формы задаются в начале `spectrum.py` (`COLOR_DEFS`, `EXTRA`) —
их легко расширить.
